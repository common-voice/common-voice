# Browser Support
The site itself should work on the following browsers:
```
Firefox >= 52,
FirefoxAndroid >= 52,
Chrome >= 55,
ChromeAndroid >= 55,
Edge >= 15,
Safari >= 10.1,
iOS >= 10.3
```

For recording your browser needs to support the [MediaRecorder](https://developer.mozilla.org/en-US/docs/Web/API/MediaRecorder)-API. Currently those are:
```
Firefox >= 29
FirefoxAndroid >= 29
Chrome >= 49
```
